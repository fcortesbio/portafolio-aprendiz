## 📂 Estructura del Portafolio

El contenido está organizado en las tres etapas de formación, siguiendo la metodología del SENA:

`etapa-induccion/`: Documentación inicial y contextualización del programa.

`etapa-analisis/`: Fundamentos del Diseño y Analis de Sistemas, diagramas UML y especificaciones técnicas.

`etapa-productiva/`: *Coming soon*

`actividades-extra/`: Quices, talleres, muestras de código, etc.

Este portafolio utiliza un flujo de trabajo moderno para garantizar la integridad de las evidencias:

- **Origen**: Notas gestionadas en [Obsidian](https://obsidian.md/) (Markdown).

- **Control de Versiones**: Repositorio privado en GitHub.

- **Espejo de Consulta**: Sincronización automática mediante [GitHub Actions](https://github.com/fcortesbio/portafolio-aprendiz/blob/main/.github/workflows/sync_to_drive.yml ) hacia Google Drive para facilitar la revisión del instructor.


**Nota para el revisor**: Si está leyendo esto en Google Drive, tenga en cuenta que el historial de versiones y el código fuente completo residen en el [repositorio de GitHub](https://github.com/fcortesbio/portafolio-aprendiz) vinculado a este proceso.
